Sugeng rawuh ing Website Dolanan Tradisional Nusantara! Dolanan tradisional minangka warisan budaya bangsa ingkang kawariske turun-temurun lan gadhah teges wigati tumraping gesang sosial masyarakat Indonesia. Dolanan punika boten namung dados sarana hiburan, nanging ugi minangka media pawiyatan ingkang maringi piwulang bab ketrampilan, kerjasama, kreativitas, kecerdasan, lan rasa kebersamaan. Ing jaman rumiyin, dolanan tradisional dados péranganing kathahing bocah lan asring dipunlampahi kanthi alami ing halaman griya, lapangan désa, lan papan-papan tebuka sanèsipun. Website punika dipunwujudaken kanggé ngenalaken malih manéka warni dolanan tradisional dhumateng para mudha, nglestantunaken ajining budaya ingkang kapendhet ing salebeting dolanan, saha nyawisaken katrangan lengkap babagan jinis-jinis dolanan. Saben dolanan gadhah sajarah, piranti, lan cara main ingkang béda-béda sarta ngemot nilai edukatif ingkang wigati kanggé ngéwahi perkembangan jasmani lan rohani para bocah. Mugi kanthi anané website punika, dolanan tradisional saged dipunkenali malih, dipunregani, lan dipunlampahi déning masyarakat Indonesia.

Manfaat Dolanan Tradisional:
1. Dolanan tradisional maringi kathah paédah, antawisipun:
2. Ngéwahi motorik kasar lan motorik alit,
3. Nglatih kerjasama lan kekompakan,
4. Maringi piwulang sportifitas lan disiplin,
5. Ningkataken kreativitas lan kacepetan mikir,
6. Ngreketaken paseduluran antarpamaén,
7. Njagi kasehatan jasmani awit kathah gerakan ingkang migunani.
